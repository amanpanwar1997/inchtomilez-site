
### Step 2: Replace Typography
```tsx

<p className="text-base text-gray-700">Text</p>

// ‚úÖ NEW
<Heading as="h2" variant="lg">Title</Heading>
<BodyText variant="md">Text</BodyText>

// OR use utility classes

```

### Step 3: Replace Buttons
```tsx
// ‚ùå OLD

  Click Me
</button>

// ‚úÖ NEW
<Button variant="primary" size="md">
  Click Me
</Button>

// OR use utility class
<button className="btn-award">
  Click Me
</button>
```

### Step 4: Replace Colors
```tsx
// Search and replace these patterns:

bg-[#baff00]          ‚Üí bg-black
bg-[#ffc60b]          ‚Üí bg-black
text-[#baff00]        ‚Üí text-black OR text-[#404040]
text-green-500        ‚Üí text-black
hover:bg-[#baff00]    ‚Üí hover:bg-[#404040]
border-[#baff00]      ‚Üí border-black
```

### Step 5: Update Border Radius
```tsx
// ‚ùå OLD
rounded-xl
rounded-2xl
rounded-3xl

// ‚úÖ NEW - Everything uses 10px
rounded-[10px]
```

### Step 6: Add Glassmorphism
```tsx

// ‚úÖ NEW
<Card variant="card" padding="md" hover>
```
